# Employee Management System

A modern, role-based **Employee Management System** built using **React.js** and **Tailwind CSS**, featuring separate **Admin** and **Employee** panels with **LocalStorage-based data persistence**.

This project demonstrates clean frontend architecture, responsive UI design, and real-world admin dashboard functionality without using a backend.
